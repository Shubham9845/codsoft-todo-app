# ✅ To-Do List App — CodSoft Python Project

A smart To-Do List web application built with Python and Streamlit.

## 🌐 Live Demo
https://codsoft-todo-app-gr3il7tptfsqq9bv8xrtxz.streamlit.app

## 📌 Features
- Add tasks with priority levels (High, Medium, Low)
- Mark tasks as completed
- Delete individual tasks
- Progress tracker
- Mark all tasks done at once
- Clear all tasks

## 🛠️ Tech Stack
- Python
- Streamlit
- Session State Management

## 🚀 Run Locally
pip install streamlit
streamlit run app.py
